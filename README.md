The goal of our project was to create a simplified version of the game "Stack" in Matlab allowing for user input to determine the outcome of the game (win or lose).

In our version of "Stack", the "blocks" are two-dimensional rectangles, where the user presses the space bar in order to "stack" the blocks on top of one another. If the stacked block is not perfectly aligned with the block below it, the overlapping portion will be removed and the subsequently generated block will be created at that size. However, if the user is able to make it to the top of the screen without the block falling to the bottom, they win! Whereas if they do not, they lose.  
The format of game-play to win was made to occur once enough rectangles have been stacked to the top of the predetermined game screen. In order to play the stack game, each time the space key is pressed, the current rectangle will drop onto the stack.

Our code was all implemented and displayed under the file name "stack_game2.m", where other files listed either reset/play coding or exist as a backup in order to commit previous codes.

We hope you enjoy playing our stack game! (Press Space to "Stack")
